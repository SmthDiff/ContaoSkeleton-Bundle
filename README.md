# Contao Skeleton Bundle

This contao module is a template for building our own.

## Requirements

- Contao 4.13+
- PHP 7.4 or 8.0+

## Install

!!! example installation !!!

```BASH
$ composer require smthdiff/contaoskeleton-bundle
```
